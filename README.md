# Discord Spammer  
Made for easy MP spamming on Discord ! WARNING : you may be reported !  
***  
## How to use :  
![window.png](https://i.ibb.co/pwjrjNg/window.png)  
- 1: The text to be spammed  
- 2: The delay in milisecond between each messages.  
> Don't set it to 0, it will crashes !  
- 3: The number of times the message will be send.  
> If you check the fourth option, one message means all the letters of the text !  
- 4: When checked, all of the letters will be spammed.  
> Example :  
> T  
> e  
> s  
> t  
> Warning : This option is a snapshot and may miss some letters !

## Building Discord Spammer
You can build Discord Spammer yourself.  
Follow these simple steps :  
- Download [the main source zip](https://sinikraft.github.io/website/magic/dllatest?repo_link=https://api.github.com/repos/SiniKraft/Discord-Spammer/releases/latest)
- Extract the zip
- Install python 3.6 +
- Run command 'python -m pip install -r requirements-win64.txt' for windows,  
- 'python3 -m pip install -r requirements-lin64.txt' for linux
- 'python -m pip install -r requirements-osx.txt' for mac os.
- Then run 'python DiscordSpammer.py'
- If you have any problem, post an issue !
